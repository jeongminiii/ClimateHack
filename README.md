## 📌 프로젝트 개요

미세플라스틱은 눈에 보이지 않지만  
하천을 통해 바다로 유입되어 **해양 생태계와 인체 건강에 장기적인 영향을 미치는 환경 문제**입니다.

본 프로젝트는 다음 질문에서 출발했습니다.

> **“비가 올 때, 우리 지역에서 발생한 미세플라스틱은  
> 얼마나 바다로 흘러갈 위험이 있을까?”**

이를 해결하기 위해  
**기후·강우·도시화 데이터를 기반으로 한 미세플라스틱 해양유출 위험지도**를 제작했습니다.

---

## 🎯 문제 정의

- 🌍 매년 약 **800만 톤**의 플라스틱이 해양으로 유입
- 🧩 상당량이 **5mm 이하 미세플라스틱**으로 분해
- 🐟 해양 생물 섭취 → 🍽️ 먹이사슬 → 👤 인체 재유입

### 🌧️ 왜 “강우”가 중요한가?

- 도로·토양·도시 표면에 쌓인 미세플라스틱이  
  **비가 올 때 빗물과 함께 하수구·하천으로 대량 유출**
- 특히 **도시화 지역 / 산업단지 / 하천 인접 지역**에서 위험 급증

👉 본 서비스는 이러한 특성을 반영해  
**강우 영향 + 도시 환경 요인**을 핵심 위험 요소로 설정합니다.

---

## 🧪 미세플라스틱이란?

### 1️⃣ 1차 미세플라스틱

처음부터 작은 크기로 제조된 플라스틱

- 세안제·치약의 마이크로비즈
- 화장품 글리터
- 세탁 시 발생하는 합성섬유
- 산업용 연마제

### 2️⃣ 2차 미세플라스틱

큰 플라스틱이 분해되어 생성

- 페트병·비닐봉지
- 타이어 마모 입자
- 도로 노면 표시 페인트
- 어업용 그물·로프
