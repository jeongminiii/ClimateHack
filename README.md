# 경기 기후 바이브코딩 해커톤

## 🚀 빠른 시작

```bash
# 의존성 설치 (이미 설치된 경우 생략)
npm install

# 개발 서버 실행
npm run dev

```

## 📦 포함된 것

- ⚡ Vite 6 - 초고속 빌드
- ⚛️ React 18 - UI 라이브러리
- 🎨 Tailwind CSS - 유틸리티 CSS
- 🎯 TypeScript - 타입 안정성
- 🎭 Lucide Icons - 아이콘
- 💎 깔끔한 스타트업 템플릿
